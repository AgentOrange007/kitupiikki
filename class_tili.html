int Tili::ysivertailuluku ( ) const

Yhdeksännumeroinen tilinumeron vertailuluku, jonka avulla tilit saadaan oikeaan järjestykseen.

Viimeinen numero kertoo otsikkotason - 1, tileillä se on 9 Otsikkotasoja saa olla siis enintään yhdeksän
